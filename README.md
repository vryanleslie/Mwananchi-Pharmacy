# Mwananchi-Pharmacy

This is a simple React-based pharmacy management application designed for Mwananchi Pharmacy. It includes modules for inventory, patient management, prescriptions, sales, and products.

## 🔐 Login Credentials
- **Username**: `admin`
- **Password**: `admin`

## ✨ Features
- User login system
- Inventory tracking
- Patient information management
- Prescription recording
- Sales tracking
- Product listing and management

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or newer)
- npm or yarn

### Installation
```bash
npm install
```

### Run the App Locally
```bash
npm start
```
The app will be available at [http://localhost:3000](http://localhost:3000).

### Build for Production
```bash
npm run build
```

## 🌐 Deployment
